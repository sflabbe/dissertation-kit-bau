\section{System Description}
\label{sec:system_description}

The system consists of coupled components that interact through defined interfaces. The main components include:
\begin{itemize}
	\item Primary subsystem with mass $m_1$
	\item Secondary subsystem with mass $m_2$
	\item Coupling elements with stiffness $k$ and damping $c$
\end{itemize}

	\centering

	\caption{Schematic representation of the system}
	\label{fig:system_schematic}
\end{figure}

Typical parameter values are:
\begin{itemize}
	\item Primary mass: $m_1 = 1000$~kg
	\item Secondary mass: $m_2 = 100$~kg
	\item Stiffness: $k = 1 \times 10^5$~N/m
	\item Damping: $c = 1000$~Ns/m
\end{itemize}

\section{Dynamic Phenomena}
\label{sec:phenomena}

The system exhibits various dynamic behaviors depending on excitation and parameters.

\subsection{Resonance}
\label{subsec:resonance}

Resonance occurs when excitation frequency matches natural frequency. The amplification factor is:
\begin{equation}
	A_f = \frac{x_{\text{max}}}{x_{\text{static}}}
	\label{eq:amplification}
\end{equation}

\section{Problem Statement}
\label{sec:problem_statement}

\section{Assumptions}
\label{sec:assumptions}

The following assumptions are adopted:
\begin{itemize}

	\item Small displacement theory
